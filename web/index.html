GIT tuto

Créer un compte sur Github https://github.com

En parallèle, créez un dossier avec la doc voulu

Dans le dossier, clic droit pour ouvrir GIT bash


Dans l'invité de commande Git bash, commencer par initialiser votre projet avec la commande : git init

Il faut maintenant faire une "mini sauvegarde" de votre projet avec la commande : git add * (* pour tout selectionner)

Rendre la sauvegarde "officiel" avec la commande : git commit -m"message" (-m"" pour voir le message )

Si on veut cloner le projet déjà existant, la commande : git clone 'liens url'

Pour créer une branche au projet, la commande : git branch 'nomDeLaBranche'

Pour naviguer entre les branches existantes, il faut utiliser la commande : git checkout 'nomDeLaBranche'
